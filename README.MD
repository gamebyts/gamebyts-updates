# GameByts Updates

This repository contains the official distributions and updates for the GameByts Desktop application.

## About

This repository hosts binary files and updates for the GameByts application. The source code is maintained in a private repository.

## Contents

- Windows executables (.exe)
- macOS disk images (.dmg)
- Linux AppImage packages (.AppImage)
- Update files (.yml)
- Release notes

## Downloads

The latest version can be downloaded from the Releases section.

Supported platforms:
- Windows (64-bit)
- macOS
- Linux

## Automatic Updates

The GameByts application checks for updates automatically and will notify users when new versions are available.
